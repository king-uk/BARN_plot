# BARN plot

## Dependancy

 - Matplotlib
## Map Download and Modification

```bash
git clone https://github.com/king-uk/BARN_plot.git
```

## Please download BARN_dataset/grid_files from [BARN_dataset](https://www.cs.utexas.edu/~xiao/BARN/BARN.html)
```bash
cd BARN_dataset
python3 npy_to_txt.py
```

---
## Usage

```bash
python3 main.py
```
